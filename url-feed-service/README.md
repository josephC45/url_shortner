# URL-Feed-Service

URL-Feed-Service is a REST API that allows users to see the last 10 globally created URLs.

---

## Features

- Reactive REST API
- Calls Redis to get back 10 most recently created URLs
- Deserializes redis payload and returns to user

---

##  Tech Stack

- Java
- Spring Boot 3.5.1 
- Jackson (JSON)
- Redis

---

### Prerequisites

- Redis
- Docker (optional)
